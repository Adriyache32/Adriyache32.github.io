El Server

Versión 1.20.4 - 1.21
Modo Survival / Hard
Slot 20 jugadores
Tipo Whitelist
Plugins VoiceChat, CoreProtect, WorldEdit
Reglas No grief / No hacks / Respeto

Server survival privado para amigos y conocidos. Ambiente tranquilo, construcción colaborativa y eventos periódicos. Todos los jugadores pasan por un proceso de whitelist para mantener la calidad.

Contamos con voice chat de proximidad, protección de construcciones con CoreProtect y un mapa dinámico para ver el mundo desde el navegador.

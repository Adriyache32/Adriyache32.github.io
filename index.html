El Server

Versión1.20.4 - 1.21
ModoSurvival / Hard
Slot20 jugadores
TipoWhitelist
PluginsVoiceChat, CoreProtect, WorldEdit
ReglasNo grief / No hacks / Respeto

Server survival privado para amigos y conocidos. Ambiente tranquilo, construcción colaborativa y eventos periódicos. Todos los jugadores pasan por un proceso de whitelist para mantener la calidad.

Contamos con voice chat de proximidad, protección de construcciones con CoreProtect y un mapa dinámico para ver el mundo desde el navegador.

Guía para Jugar

/// Todo lo que necesitás para conectarte ///

1

Descargá SKLauncher

SKLauncher es un launcher offline — no necesitás cuenta de Microsoft ni premium.

Descargar SKLauncher
2

Instalá SKLauncher

Ejecutá el archivo descargado. Te va a pedir una versión de Java si no la tenés — seguí las instrucciones.

Creá un perfil con el nombre que quieras (no necesita email real).

3

Instalá los Mods

Descargalos de la sección de arriba. Poné los archivos .jar en la carpeta mods de tu instancia de Minecraft.

Ruta típica: %appdata%/.minecraft/mods/ o en SKLauncher: sklauncher/minecraft/mods/

4

Conectate al Server

Abrí Minecraft, andá a Multijugador y agregá nervalia.mc. Necesitás whitelist — pedila en nuestro Discord.

⚠ No necesitás cuenta premium. Modo offline total.

+

Shaders & Texture Packs

Los shaders van en shaderpacks/ y las textures en resourcepacks/ dentro de la misma carpeta de Minecraft.

Activarlos desde el juego: Opciones > Paquetes de recursos / Sombras.

FAQ

/// Preguntas frecuentes ///

¿Cómo entro al server?
Primero tenés que pedir whitelist a algún miembro del equipo. Una vez agregado, conectate con la IP nervalia.mc.
¿Qué versión necesito?
El server corre en 1.20.4 - 1.21. Cualquier versión entre esas funciona.
¿Cómo consigo monedas?
Completando logros del server. Cada logro completado te da 0.5 🪙.
¿Se puede hacer grief?
No. El server tiene whitelist y CoreProtect. Todo se revierte y el responsable es baneado.
¿Hay voice chat?
Sí, usamos Simple Voice Chat con proximidad. Descargalo de la sección Mods.

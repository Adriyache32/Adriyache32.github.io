El Server

Versión1.20.4 - 1.21
ModoSurvival / Hard
Slot20 jugadores
TipoWhitelist
PluginsVoiceChat, CoreProtect, WorldEdit
ReglasNo grief / No hacks / Respeto

Server survival privado para amigos y conocidos. Ambiente tranquilo, construcción colaborativa y eventos periódicos. Todos los jugadores pasan por un proceso de whitelist para mantener la calidad.

Contamos con voice chat de proximidad, protección de construcciones con CoreProtect y un mapa dinámico para ver el mundo desde el navegador.

Reglas del Server

/// Mantengamos el orden ///

01
No Griefing

No destruir construcciones ajenas sin permiso.

02
No Hacks

Prohibidos clientes modificados con ventajas injustas.

03
Respeto

Trata a todos con respeto. No toleramos toxicidad.

04
No Robar

No tomar items de cofres o bases ajenas sin permiso.

05
Duplicación

Prohibido duplicar items o usar bugs intencionalmente.

06
Voice Chat

Uso obligatorio de voice chat de proximidad para mejor experiencia.
